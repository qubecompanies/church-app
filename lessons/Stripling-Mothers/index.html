i First-time Firebase setup?
This file requires a Firebase Firestore project to sync between devices. If you haven't yet:
1. Open the file in a text editor
2. Find FIREBASE_CONFIG near the top of the script
3. Paste your Firebase project's config object (from Firebase console → Project settings → Your apps → Config)
4. In Firestore Rules, allow reads/writes on /sessions/{room}
Save and reload. One-time setup — works for every Mother's Day after.
